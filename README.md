# Student Academic Performance Analysis System

## Project Overview
This project implements a comprehensive system for managing and analyzing student academic data using Python.

## Features
1. **Data Management**: Stores student profiles using dictionaries, lists, sets, and tuples
2. **Grading System**: Converts scores to grades with feedback using match-case
3. **Interactive Menu**: 5-option console interface for user interaction
4. **Eligibility Checker**: Determines graduation eligibility using logical operators
5. **Data Analysis**: Set operations, list slicing, and nested data processing
